== Description ==

The Set.fm Wordpress Widget enables you to display your most recent set or sets from your Wordpress sidebar.  

== Installation ==

1. Place `setfm-widget` in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login to your plugin from the 'Plugins' menu in WordPress
